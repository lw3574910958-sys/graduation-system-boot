# 枚举类模块化分类重构说明

## 重构背景
为提高代码组织性和可维护性，按照功能模块对枚举类进行分类整理，使项目结构更加清晰。

## 分类方案

### 1. 用户相关 (user/)
- **AccountStatus.java** - 账户状态枚举
- **Gender.java** - 性别枚举  
- **UserType.java** - 用户类型枚举

### 2. 权限相关 (permission/)
- **Permission.java** - 系统权限枚举
- **SystemRole.java** - 系统角色枚举
- **AdminRoleLevel.java** - 管理员角色等级枚举

### 3. 业务状态 (status/)
- **ReviewStatus.java** - 审核状态枚举
- **SelectionStatus.java** - 选题状态枚举
- **TopicStatus.java** - 题目状态枚举

### 4. 文档相关 (document/)
- **FileType.java** - 文件类型枚举

### 5. 成绩相关 (grade/)
- **GradeLevel.java** - 成绩等级枚举

### 6. 通知相关 (notice/)
- **NoticeStatus.java** - 通知状态枚举
- **NoticeType.java** - 通知类型枚举

## 导入路径变更

### 旧导入路径
```java
import com.lw.graduation.domain.enums.XXX;
```

### 新导入路径
```java
// 用户相关
import com.lw.graduation.domain.enums.user.AccountStatus;
import com.lw.graduation.domain.enums.user.Gender;
import com.lw.graduation.domain.enums.user.UserType;

// 权限相关
import com.lw.graduation.domain.enums.permission.Permission;
import import com.lw.graduation.domain.enums.permission.SystemRole;
import import com.lw.graduation.domain.enums.permission.AdminRoleLevel;

// 业务状态
import com.lw.graduation.domain.enums.status.ReviewStatus;
import import com.lw.graduation.domain.enums.status.SelectionStatus;
import import com.lw.graduation.domain.enums.status.TopicStatus;

// 文档相关
import com.lw.graduation.domain.enums.document.FileType;

// 成绩相关
import com.lw.graduation.domain.enums.grade.GradeLevel;
```

## 影响范围

本次重构涉及以下模块的导入语句更新：
- graduation-domain (BizGrade.java)
- graduation-document (DocumentServiceImpl.java)
- graduation-selection (SelectionServiceImpl.java)
- graduation-topic (TopicServiceImpl.java)
- graduation-user (UserServiceImpl.java, UserRoleService.java, UserRoleServiceImpl.java)
- graduation-grade (GradeServiceImpl.java, GradeCalculatorServiceImpl.java)

## 验证结果

✅ 所有枚举类已按功能模块正确分类
✅ 相关导入语句已更新
✅ 项目编译通过，无错误
✅ 功能不受影响

## 后续维护建议

1. 新增枚举类时，应根据其功能归属选择合适的模块目录
2. 跨模块引用时注意使用完整的包路径
3. 保持各模块内枚举类的相关性和内聚性
